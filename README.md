## Description
Csed is a simple C# terminal text editor inspired by Vim.

## Dependencies
* dotnet-sdk-8.0

## Usage
```bash
csed file.txt
```

## Build Instructions
To begin, clone the project and go to the root of the repository:
```bash
git clone https://github.com/BlinkDynamo/Csed.git
cd Csed/
```

Build the project with dotnet:
```bash
dotnet build
```
